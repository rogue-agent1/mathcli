# mathcli
CLI calculator: eval, stats, base conversion, percentages, primes, fibonacci.
```bash
python mathcli.py eval "sqrt(2) * pi"
python mathcli.py stats 1 2 3 4 5
python mathcli.py base 0xff --from-base 16
python mathcli.py percent 15 of 200
python mathcli.py prime 97
python mathcli.py fib 20
```
## Zero dependencies. Python 3.6+.
